# Enterprise Active Directory Lab

## Project Overview

This project demonstrates the deployment and administration of a Windows Server Active Directory environment for a simulated organization.

## Objectives

- Deploy Active Directory Domain Services
- Configure DNS
- Configure DHCP
- Create Organizational Units
- Create Users and Groups
- Implement Group Policies

## Technologies Used

- Windows Server 2022
- Active Directory
- DNS
- DHCP
- Group Policy Management

## Skills Demonstrated

- Identity and Access Management
- User Provisioning
- Group Policy Administration
- DNS Administration
- Windows Server Administration
